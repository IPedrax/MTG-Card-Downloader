Format Guide:
Quantity
Optional: Number of cards (e.g., 4). Defaults to 1 if omitted.
Card Name
Full card name, can include spaces (e.g., Lightning Bolt or Tribute to Horobi / Echo of Death's Wail)
Set Code & CN
Optional: Set code in parentheses and Collector Number (e.g., (M10) 146). If omitted, the default print will be downloaded (Fallback).
Finish
Optional: Foil (*F*) or Etched (*E*) marker.
Double-Sided
Cards with two faces will have both sides downloaded to a DS folder within the ZIP.
